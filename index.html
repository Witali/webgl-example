WebGL Image Decode Demo

Small demos for the textured cube, JPEG decoding, and WebP decoding experiments.

Cube Rotating WebGL cube with the decoded stone texture. BPAL texture sampler Sample BPAL mip levels in a shader with bilinear, trilinear, or anisotropic filtering. Image decoding Compare browser output with JPEG GPU/WASM paths and WebP WASM. Image palette Reduce an image to a selectable palette with weighted OKLab or RGB k-means clustering. Block palette compression Compress blocks with local color indices that reference one shared image palette. Retro image converter Export ZX Spectrum .scr screens and planar PC VGA Mode X image data. Benchmarks Run native browser, JPEG GPU/WASM, and WebP WASM decode timings. Browser capabilities Inspect WebGPU, WebGL, and WebAssembly support in this browser.
